Posicionamentos

P1 O posicionamento em CSS permite controlar onde os elementos aparecem na página. Existem quatro tipos principais: static (padrão), relative (relativo à posição original), absolute (relativo ao elemento pai posicionado) e fixed (fixo na tela). Cada tipo oferece diferentes possibilidades para criar layouts dinâmicos e responsivos.
